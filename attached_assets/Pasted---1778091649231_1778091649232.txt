지시문

상품/단가 관리의 “통역장비” 유형을 언어 기반 구조가 아닌 “장비상품 기반 구조”로 수정한다.

현재 문제:

* 통역장비 선택 시에도 출발언어/도착언어가 표시됨
* 통역장비는 언어 기반 상품이 아니라 장비 카탈로그형 상품임
* 실제 사용중인 통역장비 엑셀 기준과 현재 UI 구조가 맞지 않음

목표:
통역장비는 번역/통역과 완전히 다른 구조로 동작하도록 수정.

━━━━━━━━━━━━━━━━━━

1. 통역장비 선택 시 언어 선택 제거
   ━━━━━━━━━━━━━━━━━━

상품유형이:

* 통역장비

일 경우:

* 출발언어 숨김
* 도착언어 숨김

처리.

검색 필터와 상품 등록/수정 화면 모두 동일 적용.

━━━━━━━━━━━━━━━━━━
2. 통역장비 전용 “장비상품” 선택 구조 추가
━━━━━━━━━━━━━━━━━━

상품유형 = 통역장비 선택 시
언어 영역 대신 아래 표시:

장비상품

searchable select 구조 사용.

━━━━━━━━━━━━━━━━━━
3. 현재 사용중인 장비 상품 기본 등록
━━━━━━━━━━━━━━━━━━

현재 엑셀 기준 장비상품들을 기본 옵션으로 등록.

예:

* 동시통역장비(FM)
* 동시통역장비(적외선)
* 리시버
* 리시버 분실 보상비

* 통역부스
* 사전설치
* 가이드장비(수신기)
* 가이드장비(송신기)
* 위스퍼링장비(수신기)
* 위스퍼링장비(송신기)
* 회의용마이크
* 무선마이크
* 유선마이크
* 핀마이크
* 오디오인터페이스
* 음향콘솔
* 믹서
* 분배기

등.

━━━━━━━━━━━━━━━━━━
4. 장비상품 searchable select 적용
━━━━━━━━━━━━━━━━━━

장비상품은:

* 검색 가능
* 스크롤 가능
* 긴 리스트 대응 가능

구조 사용.

━━━━━━━━━━━━━━━━━━
5. 기타 직접입력 지원
━━━━━━━━━━━━━━━━━━

장비상품에서도:

* 기타 직접입력

선택 가능.

선택 시:

* 직접 입력 장비상품 input 생성.

예:
직접 입력 장비상품
[                    ]

━━━━━━━━━━━━━━━━━━
6. 상품 리스트 표시 수정
━━━━━━━━━━━━━━━━━━

통역장비 상품은:

* 언어 표시 제거

예:
한국어 → 영어

같은 언어 태그 제거.

대신:

* 장비상품명
* 단가
* 단위

중심 표시.

━━━━━━━━━━━━━━━━━━
7. 검색 필터 수정
━━━━━━━━━━━━━━━━━━

상품유형 = 통역장비 선택 시:

표시:

* 장비상품 필터

숨김:

* 출발언어
* 도착언어

━━━━━━━━━━━━━━━━━━
8. 기존 데이터 유지
━━━━━━━━━━━━━━━━━━

기존 통역장비 상품 데이터 손실 없이 유지.

기존 상품이:

* 통역장비
* 언어값 포함

되어 있어도 조회 오류 없이 처리.

━━━━━━━━━━━━━━━━━━
9. dropdown UX 개선
━━━━━━━━━━━━━━━━━━

현재:
상품유형 또는 장비상품 dropdown을 열고 캡쳐하려 하면 즉시 닫힘.

수정:

* searchable popover 구조 사용
* outside click 처리 개선
* 캡쳐/검토 가능한 수준으로 유지

10. E2E 확인
━━━━━━━━━━━━━━━━━━

반드시 확인:

[검색 필터]

* 상품유형 = 번역 → 언어 필터 표시
* 상품유형 = 통역 → 언어 필터 표시
* 상품유형 = 통역장비 → 언어 필터 숨김 + 장비상품 필터 표시

[상품 등록]

* 통역장비 선택 시 언어영역 숨김
* 장비상품 select 표시
* 장비상품 검색 가능
* 기타 직접입력 가능

[상품 리스트]

* 통역장비 상품에서 언어 태그 제거
* 장비상품 중심 표시

[기존 데이터]

* 기존 통역장비 상품 정상 조회
* 기존 번역/통역 상품 영향 없는지 확인

E2E 결과까지 보고.
